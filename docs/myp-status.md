## Props

| Prop name | Description | Type   | Values | Default |
| --------- | ----------- | ------ | ------ | ------- |
| bgType    | 背景主题    | string | -      | ''      |
| boxStyle  | 外层样式    | string | -      | ''      |

## Slots

| Name    | Description | Bindings |
| ------- | ----------- | -------- |
| default |             |          |

---

myp-status 为您提供了快速决策 statusBar 高度的能力。您可以在需要使用 statusBar 高度的地方快速调用。

## bgType

主题背景色。`myp-bg-` 下定义的可以使用的主题色。

## boxStyle

个性化的style设置。
